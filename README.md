# Bruce-JS-Apps

JavaScript apps for [Bruce Firmware](https://github.com/pr3y/Bruce) devices, published on the [Bruce App Store](https://github.com/BruceDevices/App-Store-Data).

**Author**: [@Senape3000](https://github.com/Senape3000)

---

## Apps

### ProtoPirate v1.0.0 — RF
Car key fob decoder and emulator with 16 protocols, full crypto suite (KeeLoq, TEA, PSA XOR), 9 emulatable protocols with counter advance and button remapping.

- **Category**: RF
- **Supported Devices**: Lilygo T-Embed CC1101, Lilygo T-Embed CC1101 Plus
- **[Details](ProtoPirate/README.md)**

### BruteRF v1.0.0 — RF
Complete RF brute-force tool with 34 protocols, 7 categories, De Bruijn attack, RAW brute-force, pause/replay mode, .sub file saving. Responsive UI for all screen sizes.

- **Category**: RF
- **Supported Devices**: All (responsive layout for 6 screen resolutions)
- **[Details](BruteRF/README.md)**

---

## Installation

Install directly from the **Bruce App Store** on your device, or copy the `.js` files to your SD card under `/BruceJS/RF/`.
